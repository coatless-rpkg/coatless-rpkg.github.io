# coatless-rpkg.github.io

Auto-built project index for [coatless-rpkg](https://github.com/coatless-rpkg), deployed to
[r-pkg.thecoatlessprofessor.com](https://r-pkg.thecoatlessprofessor.com/).

Source: [coatless/quarto-org-index](https://github.com/coatless/quarto-org-index).

Rebuilt daily by GitHub Actions plus on push to `main` and on
`repository_dispatch` event `refresh`.
